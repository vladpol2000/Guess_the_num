# Guess The Number (CLI Game in Go)

Консольная игра на Go, в которой пользователь пытается угадать случайное число за ограниченное количество попыток.

---

## Установка и запуск

```bash
git clone https://github.com/your-username/guess-the-number.git
cd guess-the-number

go mod tidy
go run main.go

---

## Описание игры

Консольная игра на Go, в которой пользователю нужно угадать случайно сгенерированное число за ограниченное количество попыток. В проекте реализованы уровни сложности: Easy (1–50, 15 попыток), Medium (1–100, 10 попыток) и Hard (1–200, 5 попыток). Игра включает систему подсказок и обработку пользовательского ввода с валидацией через bufio.Scanner, что обеспечивает стабильную работу с stdin. Результаты каждой игры сохраняются в файл result.json в формате JSON.
